15. Criando meu primeiro módulo Python (*.py)

Pastas de projetos

Nas pastas criadas para o projeto em python, não se deve colocar espaços ou acentos no nome, sempre letras e um número.

  • É permitido colocar "_" no lugar do espaço.
  • Proibido iniciar o nome com número, sempre se inicia com letra.
